
fir_ram - фильтр с конечной импульсной характеристикой,
работающий с 1 умножителем и для хранения коэффициентов и 
данных используются блоки памяти. Производит расчет выхода 
фильтра за число тактов равное отношению порядка фильтра к
количеству параллельных умножителей.

fir_ram_top - топ файл фильтра.
